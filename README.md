# ChangeAccesionNum
將文字轉換成 MS SQL 可以查詢之字串格式的小工具。

# 功能說明
將字串加入單引號，並使用逗號隔開，轉換的同時複製內容至剪貼簿(ChangeAccesionNum 功能可用於資料庫語法)。

# 使用方式
1. 下載解壓後，開啟 ChangeAccesionNum.exe 主程式。
2. 在 Input 文字框貼入欲轉換的文字。
3. 按下「流水號轉換」按鈕，即可轉換文字，並且自動複製轉換文字於剪接簿。
